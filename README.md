# pratique-projeto1

## Como executar o arquivo .sh
- Utilize o comando (chmod 744 calculadora.sh) para conseguir tornalo executavel
- Utilize o comando (./calculadora.sh) para executar o arquivo da calculadora

## Explicando o codigo em Python
O código deste projeto e uma calculadora simples que realiza operações de soma, subtração, multiplicação e divisão. Foi criado com o intuito de aprendizagem do processo de tratamento e movimentação de arquivos em linux.
